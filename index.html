JavaScript

Termo em Inglês (Código) Tradução (Português) Significado no Contexto do Código
let Deixar / Variável Usada para declarar uma variável (local), onde você armazena valores (como a posição de um meteoro).
function Função Usada para definir um bloco de código nomeado e reutilizável, que executa uma tarefa específica.
preload Pré-carregar Função especial que é executada antes do setup. Serve para carregar recursos (como imagens e sons) para que estejam prontos antes de o programa iniciar.
setup Configuração / Preparação Função especial executada uma única vez no início do programa. Serve para configurar o ambiente (como criar a tela de desenho).
draw Desenhar / Traçar Função especial executada continuamente, em um loop. É o coração do programa, onde a lógica de atualização e desenho acontece.
if Se Inicia uma estrutura condicional, executando o código somente se a condição for verdadeira.
for Para Inicia um laço de repetição (loop) para iterar sobre uma sequência, neste caso, a lista de meteoros.
return Retornar Usado em uma função para enviar de volta um valor ou para encerrar a execução da função.
loadImage Carregar Imagem Comando que carrega um arquivo de imagem para uso no código.
createCanvas Criar Tela Comando que cria a área de desenho (tela/janela) do jogo, definindo sua largura e altura.
image Imagem Comando que desenha a imagem carregada na tela, usando coordenadas (X e Y), largura e altura.
dist Distância Comando que calcula a distância entre dois pontos (neste caso, entre a nave e o meteoro).
random Aleatório Comando que gera um valor aleatório a partir de uma faixa ou lista de opções.
constrain Restringir / Limitar Comando que limita uma variável a um valor mínimo e máximo, impedindo que ela saia desses limites.
keyPressed Tecla Pressionada Função especial que é executada toda vez que uma tecla é pressionada pelo usuário.
noLoop Sem Repetição / Parar Loop Comando que interrompe a execução contínua da função draw(), efetivamente congelando o jogo (usado no "Game Over").
